# Investigation Workspace Mockup

## Layout
- **Left Panel**: Investigation timeline (vertical)
- **Center**: Evidence collection area
- **Right Panel**: Investigation playbook steps

## Timeline View
- Chronological events with timestamps
- Color-coded event types (API calls, data access, alerts)
- Expandable event details

## Evidence Cards
- Screenshots/logs of suspicious activity
- Network traffic graphs
- Agent behavior patterns

## Playbook Steps
- Checklist of investigation steps
- Completed steps with checkmarks
- Current step highlighted
- Next steps grayed out
