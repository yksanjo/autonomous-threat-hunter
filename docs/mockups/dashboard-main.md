# Main Dashboard Mockup Specification

## Layout
- **Header**: Dark theme (#1E1E1E background)
- **Top Bar**: Logo left, user menu right, notifications icon
- **Sidebar**: Navigation menu (Threats, Agents, Investigations, Responses, Analytics)
- **Main Content**: 3-column grid layout

## Key Metrics Cards (Top Row)
1. **Active Threats**: Large number, red indicator, trend arrow
2. **MTTD**: <1 minute, green badge
3. **MTTR**: <5 minutes, green badge
4. **Detection Rate**: 99.2%, green progress bar

## Threat Timeline (Center)
- Interactive line chart showing threats over last 24 hours
- Color-coded by severity (red=critical, yellow=high, blue=medium)
- Hover tooltips with threat details

## Agent Inventory (Right Panel)
- List of agents with health indicators
- Risk scores (0-100) with color coding
- Quick filter buttons

## Active Investigations (Bottom)
- Card-based layout showing ongoing investigations
- Progress indicators
- Time elapsed badges
